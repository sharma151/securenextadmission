# Secure Next Admission

A professional, high-performance admission portal built with **Next.js**, **Tailwind CSS**, and **Shadcn UI**. This project is designed to streamline the college inquiry and enrollment process with a focus on speed and user experience.

---

## рџљЂ Tech Stack

- **Framework:** [Next.js 14/15 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Shadcn UI](https://ui.shadcn.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Icons & Assets:** Optimized WebP imagery and custom SVG icons.

## рџљЂ Quick Start

Run the following commands to get the project up and running locally:

```bash
# Clone the repository
git clone [https://github.com/sharma151/securenextadmission.git](https://github.com/sharma151/securenextadmission.git)

# Enter the directory
cd securenextadmission

# Install dependencies
yarn install

# Start the development server
yarn dev

```
